1. Introduction

These instructions will guide you through a test build of the FieldWorks database files to verify that all is setup properly.

2. Compilation

  • Launch the VC++ command window found in C:\VCS\Wycliffe\projects\FieldWorks\Src\Sql\Scripts.

  • There are two scripts

    1. testbuild.bat - used for test only purposes while the development work is in progress. Note that the default path for the Firebird base folder was modified to be C:\Firebird to simplify development.
    2. build.bat - used to build the final and complete version

  • Ensure to revise the Firebird reference paths in testbuild.bat if Firebird isn't installed at C:\Firebird on your system. For example if you've installed Firebird at E:\Firebird then tesbuild.bat would be revised as follows:

    set FBBIN=E:\Firebird\bin
set FBDB=E:\Firebird\Data\FieldWorks\FW01.DB

  • Type testbuild.bat at the prompt.

  • Click on the following link to see a typical output results of running testbuild.bat where there are no errors: testbuild.txt
